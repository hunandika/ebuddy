# Ebuddy Next Typescript App

## 📌 Description
This project is a Next.js-based application, a React framework that enables server-side rendering and fast, efficient web app development.

## 🚀 Installation and Running the Project

### 1. Clone the Repository
```sh
git clone git@github.com:hunandika/ebuddy.git
cd ebuddy
```

### 2. Install Dependencies
```sh
npm install
# or
yarn install
```

### 3. Run the Application
```sh
npm run dev
```
The application will run at `http://localhost:3000`

## 📂 Directory Structure
```
.
├── apis/          # Helper API
├── app/           # Main application app router
├── components/    # Reusable atomic components
├── config/        # Static Config
├── theme/         # Theme config MUI
├── public/        # Static files like images and icons
├── store/         # Redux store
├── .env.local     # Environment configuration
└── next.config.js # Next.js configuration
```

## 🛠 Build & Deployment
To build the project:
```sh
npm run build
```
To run the built project:
```sh
npm start
```

## 🛠️ Technologies Used

- **Next Typescript**: Frontend framework app router.
- **Mui**: Reusable component provided by MUI.
